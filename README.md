## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

bash
$ npm install


## Compile and run the project

bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod


## Create file .env
```
DB_HOST
DB_PORT
DB_USER
DB_PASS
DB_NAME
```
